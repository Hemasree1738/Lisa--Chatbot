📌 Description

This is a beginner-friendly chatbot built using Python. The chatbot interacts with users through text input and can respond to basic commands such as greetings, time queries, mood detection, and even tell jokes to cheer the user up.

🚀 Features
👋 Greets the user and asks for their name
💬 Responds to basic inputs like:
"hello", "how"
"bye"
"time"
"age"
😊 Detects if the user is feeling sad
😂 Tells random jokes to improve mood
⏰ Displays current time
🔁 Runs in a loop until the user exits
🛠️ Technologies Used
Python
Built-in modules:
random
datetime
▶️ How to Run
Make sure Python is installed on your system

Clone this repository:

git clone https://github.com/Hemasree1738/Lisa--Chatbot.git

Navigate to the project folder:

cd chatbot-project

Run the script:

python chatbot.py
